# بوابة إدارة النظام

واجهة تجريبية بسيطة لإدارة تسجيل الدخول ولوحة المشرف مع إدارة الصلاحيات.

## المتطلبات

- متصفح حديث يدعم JavaScript.
- خادم محلي بسيط لتشغيل الملفات (اختياري لكن موصى به لتفادي قيود المتصفح على `fetch`).

## التشغيل محليًا

```bash
# باستخدام Python
python -m http.server 5173
```

ثم افتح المتصفح على:

```
http://localhost:5173/src/index.html
```

## البناء

لا يوجد بناء مخصص، حيث يعتمد المشروع على ملفات HTML/CSS/JS خام.

## بيانات تسجيل الدخول التجريبية

- اسم المستخدم: `admin`
- كلمة المرور: `Admin@2024`

يمكن تعديل هذه البيانات في `src/data/admin.json`.
